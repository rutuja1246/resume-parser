# Automated Resume Parser

A Flask-based web application that extracts information from PDF and DOCX resumes using Natural Language Processing (spaCy) and stores candidate details in a PostgreSQL database.

## Features

- Upload PDF/DOCX resumes
- Extract Name, Email, Phone
- Extract Skills
- Extract Education
- Extract Experience
- Store candidate data in PostgreSQL
- Search candidates by skill

## Tech Stack

- Python
- Flask
- spaCy
- pdfplumber
- python-docx
- SQLAlchemy
- PostgreSQL
